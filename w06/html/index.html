World Wide Web

흔히 WWW, W3, Web 불리는 World Wide Web은 인터넷을 통해 접근 가능한 공용 웹페이지의 상호연결 시스템입니다. 웹은 인터넷과 다르며, 인터넷을 기반으로 한 수많은 응용 프로그램 중 하나입니다.

오늘날 우리가 "웹"이라고 알고 있는 시스템은 다양한 구성요소로 이루어져 있습니다.

웹의 개념은 하이퍼링크로 링크되고 연결된 문서를 보면 잘 알 수 있는데, 웹의 정체성은 연결된 문서들의 집합으로서 정의됩니다.
